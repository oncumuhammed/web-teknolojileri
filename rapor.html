📄 Proje Sayfaları ve Ekran Görüntüleri

Projenin her menü sayfasına ait ekran görüntüsü ve kısa açıklaması aşağıda sunulmuştur.

1. Hakkında (Giriş) Sayfası index.html

Sayfa sahibini tanıtan kişisel bilgiler, motto, ilgi alanları (Vücut Geliştirme, IoT, Otomasyon), medya öğeleri (profil resmi ve YouTube video embed) ve diğer sayfalara yönlendiren bağlantılar içermektedir. Responsive tasarım Bootstrap 5 grid altyapısıyla sağlanmıştır.

HTML5 CSS3 Bootstrap 5 Responsive YouTube Embed

2. Özgeçmiş (CV) Sayfası cv.html

Eğitim bilgileri, danışmanlık deneyimi, yazılım/otomasyon projeleri ve yeteneklerin (Yapay Zeka, Web & Mobil, Sistem & Otomasyon kategorilerinde) yer aldığı özgeçmiş sayfasıdır. Semantik HTML5 etiketleri (header, section, article, aside) kullanılarak yapılandırılmıştır.

Semantic HTML5 header / section / article / aside Responsive Grid

3. Şehrim Sayfası sehrim.html

Hatay'a ait tanıtıcı bilgiler (nüfus, bölge, plaka, kuruluş tarihi, gezilecek yerler) ve 6 resimden oluşan tıklanabilir bir Vanilla JavaScript slider (carousel) içermektedir. Kültürel miras bölümünde Habib-i Neccar Camii ve Sen Piyer Kilisesi HTML elemanlarıyla detaylandırılmıştır.

4. Mirasımız miras.html

Hatay'ın en önemli kültürel mirası olan Habib-i Neccar Camii'ni detaylıca ele alan sayfadır. Tarihçe, mimari özellikler ve hoşgörü mirası bölümleri figure, figcaption, article, aside, dl/dt/dd, blockquote ve section gibi semantik HTML5 etiketleriyle yapılandırılmıştır.

Semantic HTML5 figure / figcaption dl / dt / dd blockquote

5. Takımımız takim.html

Hatay'ın spor takımı Atakaş Hatayspor'un detaylı tanıtım sayfasıdır. Kulüp kimliği, tarihçe, Süper Lig yolculuğu, başarılar, stadyum bilgileri ve sezon tablosu HTML elemanlarıyla (table, dl, aside, article, blockquote) detaylandırılmıştır.

HTML Table aside / article blockquote

6. İlgi Alanlarım (API Entegrasyonu) ilgi-alanlarim.html

TVMaze API üzerinden siber güvenlik, dövüş sanatları ve teknoloji temalı dizi verilerinin JavaScript Fetch API ile asenkron olarak çekilip listelendiği sayfadır. Döviz veya hava durumu API'si kullanılmamıştır. Promise.all ile çoklu sorgu yapılmaktadır.

TVMaze API Fetch API Promise.all Asenkron JS

7. İletişim Sayfası iletisim.html + islem.php

Tüm form elemanlarını (text, email, telefon, select, radio, checkbox, textarea) içeren iletişim formudur. Form verileri hem Native JavaScript hem de Vue.js 3 ile iki farklı butonda ayrı ayrı denetlenmektedir (boşluk, e-mail formatı, telefon sadece rakam vb.). HTML5'in required attribute'ı tek başına kullanılmamıştır. Doğrulanan veriler islem.php sayfasında düzenli şekilde ekrana yazdırılmaktadır.
